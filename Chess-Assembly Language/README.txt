Pentru a deschide fereastra cu jocul: Se deschide fisierul CHESS.asm in notepad++
 -> Plugins -> ASM Plugin -> Run Program;


Se va implementa jocul de șah cu mutările specifice
• pentru punctaj minim deplasarea unor piese prin selectare și validare reguli
• pentru punctaj mai mare se pot implementa:
• deplasarea tuturor pieselor
• deplasare alternativă alb-negru
• posibilitatea de a lua piesa adversarului
• highlight pe piesa selectată și eventual pe pozițiile viitoare posibile
• determinare poziție șah și șah-mat
